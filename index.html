The same image below is sized to 600px x 450px and is compressed at 80% for a file size of 33.2KB. If the connection is throttled down, this image won't load until the larger image loads.

Also, with a slow connection, this text may "jump" down after the second images loads in some browsers because it wasn't assigned an aspect ratio in the CSS (this image is 4/3). It wasn't given height and width attributes in the HTML either, which would let the browser calculate the aspect ratio.
